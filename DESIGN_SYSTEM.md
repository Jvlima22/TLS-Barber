# Design System - TLS-Barber (Premium Edition)

Este documento define as diretrizes visuais para a modernização da landing page do TLS-Barber, inspirada na estética sofisticada da Vintech.

## 1. Paleta de Cores

| Cor | Hex | Uso |
| :--- | :--- | :--- |
| **Obsidian Black** | `#0A0A0A` | Fundo principal (Background) |
| **Deep Charcoal** | `#171717` | Cards e seções secundárias |
| **Blue Accent** | `#2C78B2` | CTAs, ícones e destaques (Primária) |
| **Silver Mist** | `#A3A3A3` | Textos secundários e bordas sutis |
| **Pure White** | `#FFFFFF` | Títulos e textos principais |

## 2. Tipografia

*   **Títulos:** `Inter` ou `Geist Sans` (Bold/ExtraBold) - Visual limpo e moderno.
*   **Corpo:** `Inter` (Regular/Medium) - Alta legibilidade.

## 3. Componentes Visuais

*   **Glassmorphism:** Uso de `backdrop-blur` em headers e cards.
*   **Bordas Sutis:** Bordas com `border-white/10` para separação elegante.
*   **Gradients:** Gradientes lineares sutis de `Gold` para `Transparent`.
*   **Animações:** Transições suaves de 300ms e efeitos de hover com escala leve.

## 4. Iconografia

*   Utilização da biblioteca `lucide-react` com traços finos e cores consistentes (Gold/White).

## 5. Proposta de Valor (Copywriting)

*   **Antigo:** "Escolha o plano perfeito para sua barbearia"
*   **Novo:** "Tecnologia de Elite para o Mestre da Navalha."
*   **Slogan:** "Do agendamento à gestão financeira. O sistema completo para barbearias que buscam a perfeição."
